# Documentation

This is the documentation for the Python project template.

## Getting Started

1. Create virtual environment: `python -m venv venv`
2. Activate virtual environment: `venv\Scripts\activate` (Windows) or `source venv/bin/activate` (Unix/Mac)
3. Install dependencies: `pip install -r requirements.txt`
4. Run tests: `pytest`

## Project Structure

- `src/` - Source code directory
- `tests/` - Test files
- `docs/` - Documentation
- `pyproject.toml` - Project configuration
- `requirements.txt` - Dependencies
